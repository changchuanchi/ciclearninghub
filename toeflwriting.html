Five Principles for Score 5

① 论据具体化 — 用数据和案例替代泛泛而谈

  • ❌「研究表明，运动对健康有益。」
  • ✅「发表于《临床精神病学杂志》的研究证实，30 分钟有氧运动的抗焦虑效果与抗抑郁药物相当。」
  • 原则：引用具名机构、期刊或研究者，再配合量化数据。

② 词汇升级 — 用 B2+ 词替换基础词

  • ❌ good → ✅ compelling / beneficial / transformative
  • ❌ show → ✅ demonstrate / substantiate / corroborate
  • ❌ bad → ✅ detrimental / counterproductive / untenable

③ 结构清晰 — 标准五段式的精髓

  • 引言（1段）：背景 + 清晰立场 + 预告论点
  • 论点段（2–3段）：主题句 → 论据/例证 → 小结
  • 结论（1段）：重申立场（换词）+ 升华至更大意义

④ 辩证思维 — 承认对立面让论证更可信

  • 关键词：Admittedly... / That said... / While it is true that...
  • 原则：Score 5 的文章是「有条件的支持」，而非「非黑即白」。

⑤ 结论升华 — 不要只是重复引言

  • 将论点上升到更宏观的层面：道德意义、社会影响、代际责任等
  • 原则：结论最后一句应是全文最有深度的句子。
